Building the User Interface

This section describes how to use Apple's InterfaceBuilder application to build the main window and menu of the CurrencyConverter application. Apple's tutorial describes in detail how to do this; this section is a much briefer description of the steps.

One detail worth noting is that Apple's example was written for use with the version of InterfaceBuilder shipped with Mac OS X Tiger (10.4). This HOWTO describes OpenMCL features that are available only on Mac OS X Leopard (10.5). The main purpose, then, of this section, is to describe how to build the user interface using the Leopard version of InterfaceBuilder, which differs in some details from the Tiger version.
